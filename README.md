\# Nexus AI - Basic Chatbot Task 01



A sleek, modern AI Chatbot interface built with a FastAPI backend and a responsive Tailwind CSS frontend.



\## 🚀 Features

\- \*\*Stateless \& Stateful Chat:\*\* Maintains conversation memory using a rolling history buffer.

\- \*\*Modern UI:\*\* Glassmorphism design with smooth CSS animations.

\- \*\*Markdown Support:\*\* Renders AI responses (lists, bold text) clearly.

\- \*\*Error Handling:\*\* Robust handling for API timeouts and invalid keys.



\## 🛠️ Tech Stack

\- \*\*Backend:\*\* Python, FastAPI, Uvicorn

\- \*\*Frontend:\*\* HTML5, Tailwind CSS, JavaScript (Fetch API)

\- \*\*AI Model:\*\* Llama 3 (via Groq API)



\## ⚙️ Setup Instructions

1\. Activate Virtual Environment: `.\\venv\\Scripts\\activate`

2\. Install Dependencies: `pip install -r requirements.txt`

3\. Add API Key to `.env`: `GROQ\_API\_KEY=your\_key`

4\. Run App: `python app.py`
